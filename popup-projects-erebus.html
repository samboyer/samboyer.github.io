CMD Relay

Advanced Remote Command Execution for Windows Phone 8.1

CMD Relay is a communication system between Windows operating systems and Windows Phone 8.1, allowing remote command execution from your phone.

Through use of a WP8.1 app built from scratch, network administrators are able to effortlessly distribute code across a limitless amount of computers.

Due to limitations in the Windows Phone 8.1 Runtime API, the commands cannot be sent directly from the phone, so a relay system is utilised.

The phone connects to a specific Windows computer which is running a Relay Service and sends the payload data. Through the use of PSTOOLS, the Relay Service then sends the commands to the target computers and returns the result back to the phone.

Features:

  • Separate pages to manage Targets, Relays and Payloads
  • Send multiple payloads to multiple targets at once using the checkboxes
    • 'Send All' functionality requires press of the camera button, which increases coolness by a solid 36%
  • Multiple payload types, allowing entire executable transfer to targets
  • Targets can have a custom background colour!
  • Background Relay Service for Windows desktop operating systems
    • Runs automatically in the background with minimal impact or visibility
    • Immediately decodes and sends payload commands to target computers specified by the app
    • Fully automatic Relay service installer and uninstaller
  • Support for multiple profiles; manage different networks separately
  • Master credentials setting; set the credentials to be used for the whole network

NOTE: This application and communication system is not a hacking tool. Neither the app or the service contains methods to hack a computer system; credentials are required for the execution of any code on a target machine.
